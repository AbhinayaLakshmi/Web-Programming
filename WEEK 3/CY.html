Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style and layout web pages. CSS allows you to apply styles to web pages independently of the structure (HTML).

CSS Syntax

The syntax of CSS consists of a selector and a declaration block.

selector {

  property: value;

}

CSS Properties

CSS includes many properties. For example, the color property defines text color, and the font-size property defines the size of the text.
